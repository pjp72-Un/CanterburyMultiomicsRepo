# CanterburyMultiomicsRepo

This repository contains scripts and analysis pipelines used in a multiomics research paper. It is organised into three subdirectories, each corresponding to a distinct part of the analysis:

## Repository Structure

```
CanterburyMultiomicsRepo/
├── 16S/               # 16S rRNA amplicon sequencing analysis
├── Metabolomics/      # Metabolomics data processing and analysis
└── Multiomics/        # Multiomics integration analysis
```

## Subdirectories

### 16S
Scripts for processing and analysing 16S rRNA amplicon sequencing data, including quality filtering, denoising, taxonomic classification, and diversity analyses.

### Metabolomics
Scripts for processing and analysing metabolomics data, including pre-processing, normalisation, statistical analysis, and metabolite identification.

### Multiomics
Scripts for integrating 16S and metabolomics datasets, including correlation analyses, network construction, and joint visualisation.
