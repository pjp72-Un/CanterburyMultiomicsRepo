# Metabolomics Analysis

This directory contains scripts for processing and analysing metabolomics data.

## Contents

Scripts in this directory cover the following steps:

- **Pre-processing** – peak picking, alignment, and gap filling
- **Normalisation** – sample and feature normalisation strategies
- **Statistical analysis** – univariate and multivariate statistical tests
- **Metabolite identification** – annotation against metabolite databases
- **Pathway analysis** – enrichment analysis of metabolic pathways

## Usage

Refer to individual script headers for usage instructions and required dependencies.
