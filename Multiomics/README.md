# Multiomics Integration Analysis

This directory contains scripts for integrating 16S rRNA and metabolomics datasets.

## Contents

Scripts in this directory cover the following steps:

- **Data integration** – combining microbiome and metabolomics data matrices
- **Correlation analysis** – identifying associations between microbial taxa and metabolites
- **Network construction** – building and visualising co-occurrence / interaction networks
- **Joint dimensionality reduction** – multi-table ordination methods (e.g. DIABLO, MOFA)
- **Visualisation** – integrated plots and figures for the research paper

## Usage

Refer to individual script headers for usage instructions and required dependencies.
