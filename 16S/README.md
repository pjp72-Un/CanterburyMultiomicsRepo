# 16S rRNA Amplicon Sequencing Analysis

This directory contains scripts for processing and analysing 16S rRNA amplicon sequencing data.

## Contents

Scripts in this directory cover the following steps:

- **Quality filtering and trimming** – removal of low-quality reads and adapter sequences
- **Denoising / OTU clustering** – generating ASVs (e.g. via DADA2) or OTUs
- **Taxonomic classification** – assigning taxonomy to representative sequences
- **Diversity analysis** – alpha and beta diversity calculations and visualisation
- **Differential abundance** – identifying taxa that differ between sample groups
